# Global Conflict Monitor — Live World Map 2025

Interactive world map showing all active armed conflicts and wars as of March 2025. No login, fully open, mobile-first.

## 🚀 Deploy to GitHub Pages

1. Create a repo on GitHub (e.g. `conflict-map`)
2. Upload `index.html` to repo root
3. **Settings → Pages → Deploy from branch → main → / (root)**
4. Live at: `https://yourusername.github.io/conflict-map/`

> Only one file needed: `index.html` — everything is self-contained.

## ✨ Features
- 🗺️ **Leaflet.js interactive map** — pan, zoom, click markers
- 💥 **Pulsing animated markers** by severity (Extreme/High/Medium/Low)
- 📋 **Sidebar list** with search + severity filters
- 📰 **Detail panel** — full conflict info: deaths, displaced, parties, timeline, sources
- 📱 **Mobile-first** — bottom nav, hamburger menu, touch controls
- 🕐 **Live UTC clock**
- 🔗 **Source links** to ACLED, CFR, Crisis Group, UN
